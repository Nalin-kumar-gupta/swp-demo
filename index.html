Problem Statement

The truck loading process is a significant challenge in the logistics industry, resulting in inefficient use of space, weight imbalance, and excessive fuel consumption. This project aims to address these challenges and improve logistics efficiency.

  • NHTSA reported over 4,900 truck-related crashes in the US in 2019.
  • ATRI found the trucking industry loses approximately 1.3 billion gallons of fuel annually due to inefficient loading.

Project Overview

The Smart Walmart Packager is a solution designed to optimize trailer loading, improving package capacity, balancing axle weights, and ensuring a seamless loading experience. It provides real-time visibility into package placement and enhances the efficiency of logistics operations.

Features

  • Comprehensive Warehouse Management System
  • 3D Visualization of package placements
  • Optimized package allocation for trucks

Installation

  1. Clone the repository: git clone https://github.com/Nalin-kumar-gupta/swp-app-django.git
  2. Build the Docker image: docker-compose -f local.yml build --no-cache
  3. Start the container: docker-compose -f local.yml up

Usage

Start the application using Docker Compose and access it at localhost:3000.
